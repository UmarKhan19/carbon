# Assistant

> Bring the Carbon Assistant into Slack, with nonconformance notifications.

## Slack

Connect **Slack** over **OAuth** to put the **Carbon Assistant** in your workspace and receive
**nonconformance notifications** where your team already talks. There are no fields to fill in.

  
  ### Start the Slack install

  Authorizing opens a Slack install popup.
  
  
  ### Choose your workspace

  Pick the workspace to install into and approve the install. Carbon stores the connection. No keys to paste.
  

| Setting | What it controls |
| --- | --- |
| Connection | Established by the Slack OAuth install — no keys to paste. |

Slack is **hidden in controlled (ITAR) environments** — when Carbon runs in a controlled deployment, the
Slack integration doesn't appear at all.
