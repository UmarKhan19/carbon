# Glossary

> Every Carbon term, defined and linked: the manufacturing and accounting vocabulary the rest of the docs lean on.

Carbon names things precisely, and a few names are counterintuitive: a method is a bill of materials, work in process is a ledger balance rather than a table, and payment on an invoice is a field, not its own record. This page lists every term used across Carbon, each with the same one-line definition that powers the inline popovers. Follow **Learn more** for the full story.
